## Techniques used

    - JavaScript 
    - nodejs
    - Express 
    - PWAs 
    - indexedDB
    - Heroku-app

## Installation
1. Git clone this repository to your desktop
2. Open your coding editor and run commands
```
npm install
```
3. Start the application
```
npm start
```
4. Open your browser and visit `http://localhost:3000`
